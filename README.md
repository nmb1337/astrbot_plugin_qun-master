# 群欢迎与定时消息插件

这是一个 AstrBot 插件，提供两个功能：

1. 新成员入群欢迎（仅 OneBot v11 / aiocqhttp）。
2. 成员退群通知（仅 OneBot v11 / aiocqhttp）。
3. 群内定时发送固定消息。
4. 消息排行榜 Web 页面（默认端口 16666）。
5. 指令踢出低于指定群等级成员（先提醒，10 分钟后执行；期间发言可免踢）。
6. 提醒今天未发言成员（自动艾特 + 自定义话术）。
7. 群内指令查看当前群发言排行。
8. 可设定固定时间自动提醒当日未发言成员。
9. 定时消息支持 @全体成员。
10. 支持指令艾特群管理员，以及按天数踢出未发言成员（带白名单）。

## 功能特性

- 支持独立开关：可以单独开启或关闭入群欢迎。
- 支持群号白名单：只在指定群里触发欢迎或定时消息。
- 支持入群/退群“图片 + 文字”通知。
- 入群/退群通知采用纯文本多行展示，不再挤成单行。
- 欢迎语/退群模板支持 `\\n` 强制换行（会自动解析为真正换行）。
- 入群/退群通知默认带时间，并支持模板中使用 `{time}` 占位符。
- 入群/退群通知会自动艾特对应成员。
- 若模板未包含 `{time}`，插件会自动在末尾追加当前时间。
- 内置消息排行榜页面，可查看总榜和分群榜。
- 支持管理员指令踢出低于指定群等级成员（先艾特提醒，10 分钟后执行；观察期发言免踢）。
- 支持艾特今天未发言成员并附带自定义话术。
- 支持设置固定时间自动提醒当日未发言成员。
- 支持定时消息自动 @全体成员。
- 支持管理员一键艾特群管理员。
- 支持 `/全体成员` 指令逐个艾特群成员（不占用QQ每日 `@全体成员` 次数）。
- 支持按指定天数踢出未发言成员，并遵守踢人白名单。
- 支持群内快速查看当前群发言排行（无需访问网页）。
- 防刷屏：仅在真实入群事件触发欢迎，并做短时间去重。
- 不调用 LLM：欢迎语和定时消息均为固定模板，不消耗模型 token。
- 错误保护：发送失败会捕获异常并记录日志，不会导致插件崩溃。

## 配置项

插件配置由 `_conf_schema.json` 定义，可在 AstrBot WebUI 中配置：

- `enable_welcome`：是否启用入群欢迎。
- `welcome_group_whitelist`：允许触发欢迎的群号白名单。
- `welcome_template`：欢迎语模板，支持 `{user_id}` `{group_id}` `{time}`。
- `welcome_image`：入群通知图片（URL 或本地路径）。
- `enable_leave_notice`：是否启用退群通知。
- `leave_template`：退群通知模板，支持 `{user_id}` `{group_id}` `{operator_id}` `{sub_type}` `{time}`。
- `leave_image`：退群通知图片（URL 或本地路径）。
- `enable_rank_server`：是否启用消息排行榜 Web 服务。
- `rank_server_port`：排行榜服务端口，默认 `16666`。
- `rank_group_whitelist`：排行榜统计群白名单；为空时统计全部群。
- `kick_min_level`：`/踢低等级` 未填写等级参数时的默认等级阈值。
- `kick_user_whitelist`：踢人用户白名单（QQ号）；白名单成员不会被 `/踢低等级` 踢出。
- `kick_inactive_delay_minutes`：`/踢未发言` 的冷却分钟数（默认 `10`）。
- `speaker_stats_keep_days`：发言统计保留天数（用于 `/踢未发言` 统计窗口）。
- `enable_silent_reminder_schedule`：是否开启定时未发言提醒。
- `silent_reminder_group_whitelist`：定时未发言提醒群白名单。
- `silent_reminder_daily_time`：定时未发言提醒时间（`HH:MM`）。
- `silent_reminder_text`：`/提醒未发言` 未填写话术时的默认提醒文本。
- `silent_reminder_batch_size`：提醒时每条消息最多艾特人数。
- `enable_schedule`：是否启用定时发送。
- `schedule_group_whitelist`：允许接收定时消息的群号白名单。
- `schedule_daily_time`：每天固定发送时间（格式 `HH:MM`，24 小时制）。
- `schedule_at_all`：定时消息是否附带 `@全体成员`。
- `schedule_message`：定时发送的固定文本。

## 注意事项

- 定时消息依赖群会话缓存。插件需要先收到该群至少一条消息，才能拿到可用的会话标识并主动发送。
- 入群欢迎事件判断基于 OneBot 的 `notice_type=group_increase`。
- 如果机器人在群内被禁言或权限不足，发送会失败并写日志。
- 排行榜页面地址：`http://你的服务器IP:16666/`，JSON API：`/api/rankings`。
- 配置了 `rank_group_whitelist` 后，网页排行、API 和群内排行指令都只统计白名单中的群。
- `/踢未发言` 依赖插件已采集到的群消息统计，统计窗口由 `speaker_stats_keep_days` 控制。

## 管理指令

以下指令需要管理员权限：

- `/开启欢迎`：开启入群欢迎。
- `/关闭欢迎`：关闭入群欢迎。
- `/查看当前白名单`：查看当前入群欢迎白名单。
- `/设置欢迎语 你的模板`：设置欢迎语模板。
- `/设置入群通知 文本`：设置入群通知文字（可附带 1 张图片）。
- `/设置退群通知 文本`：设置退群通知文字（可附带 1 张图片）。
- `/添加白名单 群号`：将群号加入入群欢迎白名单。
- `/移除白名单 群号`：将群号移出入群欢迎白名单。
- `/设置未发言提醒时间 HH:MM`：设置并开启定时未发言提醒（自动将当前群加入提醒白名单）。
- `/踢低等级 等级`：先艾特提醒目标成员，10 分钟后再执行踢出；若观察期内发言则不踢；`kick_user_whitelist` 内成员始终不踢。
- `/踢未发言 天数`：先艾特提醒目标成员，按 `kick_inactive_delay_minutes` 冷却后再执行踢出；若观察期内发言则不踢（`kick_user_whitelist` 白名单成员始终不踢）。
- `/艾特管理员 附加内容`：艾特群内全部管理员/群主并附带文本。
- `/全体成员 附加内容`：逐个艾特群内成员并附带文本（避免使用 `@全体成员` 次数）。
- `/提醒未发言 自定义话术`：艾特今天未发言成员并附带话术。

以下为群内可用指令：

- `/群发言排行`：查看当前群当日发言 TOP10。

欢迎语模板支持占位符：

- `{user_id}`：新成员 ID
- `{group_id}`：群号

示例：

- `/设置欢迎语 欢迎 {user_id} 加入群 {group_id}，请先阅读群公告。`
- `/设置入群通知 欢迎 {user_id} 加入群 {group_id}`
- `/设置退群通知 成员 {user_id} 已退群，欢迎下次再来。`
- `/踢低等级 3`
- `/踢未发言 7`
- `/艾特管理员 请尽快处理入群申请`
- `/全体成员 请大家查看最新群公告`
- `/设置未发言提醒时间 21:30`
- `/提醒未发言 今天还没发言的同学，来冒个泡吧~`
- `/群发言排行`

图片+文字设置方法：

- 在群里发送上述命令时，可同时附带 1 张图片。
- 插件会提取该图片并保存为对应通知图片。
